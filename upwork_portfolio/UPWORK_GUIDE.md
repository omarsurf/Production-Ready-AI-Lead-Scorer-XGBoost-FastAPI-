## 📝 Étape 3 : Le Descriptif (Case Study)

Copie-colle ce texte sur Upwork (à adapter si besoin) :

### **Project Overview**
Sales teams often waste 70% of their time chasing leads that will never convert. This project solves that by building a custom **Machine Learning Pipeline** that ranks leads by probability of conversion *before* the first contact.

### **The Challenges**
- Handling large-scale imbalanced tabular data.
- Preventing data leakage (e.g., excluding features only known *post-call*).
- Creating a system that provides actionable "High/Medium/Low" priority rankings, not just raw scores.

### **The Solution (Tech Stack)**
- **XGBoost & Scikit-Learn**: Robust predictive modeling with custom preprocessing.
- **FastAPI**: Real-time inference server with <20ms latency.
- **Business Logic Integration**: Implementation of the **Precision@K** metric to ensure the top 10% of leads represent the highest possible conversion volume.

### **Key Results**
- **Precision@10%**: 91.8% (Meaning 9 out of 10 leads in the "High Priority" bucket are potential customers).
- **Uplift**: 4.5x improvement over random lead assignment.
- **Ready for Prod**: Integrated CLI for batch processing and REST API for CRM integration (Salesforce/HubSpot).
